## Run Locally

Clone the project

```bash
  git clone git@github.com:adrsh-raj/stocker.git
```

Go to the project directory
And Install dependencies
````
pip install -r requirement.txt
````

# give executable permission 
```bash
  chmod +x finance.py
```

Run the script 

```bash
python finance.py
```
Or
```
./script.sh
```
